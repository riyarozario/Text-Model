# Generative Text Model using GPT

This project demonstrates a **Generative Text Model** built using a **pre-trained GPT-2 architecture**.  
The model generates coherent and meaningful text based on **user-provided prompts**.  
The implementation is provided in a **Jupyter Notebook** to clearly showcase the workflow and outputs.

---

## Features
- Generates text dynamically based on **user prompts**
- Uses **pre-trained GPT-2 model** (no training required)
- Demonstrates multiple prompt examples
- Beginner-friendly implementation
- Notebook-based demonstration for easy understanding
